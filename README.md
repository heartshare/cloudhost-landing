# CloudHost Landing Page

Professional web hosting landing page with mega dropdown navigation.

## 📁 Structure

```
cloudhost-landing/
├── index.html                    # Main landing page
└── pages/
    ├── products/                  # 4 product pages
    │   ├── cloud-hosting.html
    │   ├── vps.html
    │   ├── dedicated-server.html
    │   └── cloud-storage.html
    ├── support/                   # 4 support pages
    │   ├── help-center.html
    │   ├── api-docs.html
    │   ├── system-status.html
    │   └── contact.html
    └── company/                   # 4 company pages
        ├── about.html
        ├── blog.html
        ├── careers.html
        └── contact.html
```

## 🎨 Features

- ✅ 13 complete HTML pages
- ✅ Mega dropdown navigation with Feather Icons
- ✅ Tailwind CSS (via CDN)
- ✅ Responsive design
- ✅ Interactive elements (FAQ accordion, mobile menu)

## 🚀 Quick Start

Open `index.html` in your browser.

## 📦 Deployment

Deploy to any static hosting:
- Vercel: `vercel --prod`
- Netlify: Drag and drop
- GitHub Pages: Enable in Settings

## 🛠️ Technologies

- HTML5 + Tailwind CSS (CDN)
- Feather Icons (inline SVG)
- Vanilla JavaScript
- Google Fonts (Inter)
